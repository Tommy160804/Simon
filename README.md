
Dispositivo di Test Principale: Emulatore Android Studio medium phone
Versione Android: Android 16 (API 36)

Note sulla compatibilità: L'applicazione è stata verificata anche su Emulatore Android Studio Pixel 8,
garantendo stabilità anche per android 15 (API 35).

Versione Minima: Android 7.0 (API 24).


Altre note;
- Il progetto è stato interamente realizzato usando Jetpack Compose
- L'app funziona sia in modalità portrait che landscape.
- L'app supporta 4 lingue; italiano, inglese, spagnolo, francese; la lingua del gioco cambia in base alla lingua scelta nelle impostazioni del telefono.
  Nel caso si imposti una lingua diversa da queste 4, verrà usata quella di default, ovvero l'italiano.
- La lista in ListaPartite è visualizzata in maniera inversa; in modo tale da potere vedere prima le partite più recenti.
- Per tornare a SchermataGioco vi è il bottone Nuova Sequenza
- In ListaPartite è stato usato il componente Card di Material3 per migliorare la grafica della lista.
- Il pop-up di riepilogo viene mostrato esclusivamente in caso di errore del giocatore. Se la partita viene interrotta volontariamente tramite il tasto Back o il pulsante Fine Partita, l'applicazione reindirizza il giocatore direttamente a ListaPartite senza interruzioni superflue
- L'icona dell'app è stata creata con l'AI.
- Le frequenze della classe gestioneSuoni.kt sono state generate dall'AI.
- Tutte le richieste della Consegna Finale sono state gestite.
